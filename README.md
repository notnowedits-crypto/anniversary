# 💕 Anniversary Celebration Webpage

A beautiful, romantic, and mobile-responsive anniversary webpage created with love to celebrate a special relationship milestone.

## Features

✨ **Beautiful Design**
- Cute and romantic color palette with soft pink gradients
- Smooth animations and hover effects
- Fully responsive for mobile, tablet, and desktop

📸 **Photo Gallery**
- Display beautiful photos of your loved one
- Elegant grid layout with hover animations

💫 **Live Anniversary Counter**
- Real-time counter showing days, hours, and minutes together
- Updates automatically every second

🎵 **Background Music**
- Beautiful romantic instrumental background music
- Play/pause control button
- Loops continuously

💌 **Love Message**
- Heartfelt message section
- Personalized reasons why you love them

🎀 **Animated Elements**
- Floating hearts animation
- Scroll-triggered animations
- Smooth transitions throughout

## How to Use

1. Open `index.html` in a web browser
2. The page will load with all features active
3. Click the music button (🎵) in the bottom-right to control the background music
4. Scroll through to see all the beautiful content

## Customization

You can easily customize:
- **Photos**: Replace `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg` with your own images
- **Messages**: Edit the text in the "A Message for You" section in `index.html`
- **Reasons**: Modify the "Why I Love You" cards
- **Colors**: Change the CSS variables in `styles.css`
- **Music**: Replace `background_music.mp3` with your preferred background music

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive features and counter logic
- `background_music.mp3` - Background music track
- `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg` - Photos
